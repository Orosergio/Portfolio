Input
One Telegram message with a product brief and context. No form, no UI, no setup — just chat.
AI Processing
OpenClaw parses the prompt, applies routing logic, and enriches content into structured positioning data on a self-hosted Linux VPS.
Rendering
Custom Python scripts convert markdown + JSON artifacts into a fully styled, production-ready HTML board.
Reusability
Same pipeline. Different prompt. Flower campaign → journaling app → founder copilot. Unlimited outputs.
Infrastructure
Self-hosted Hetzner VPS running OpenClaw, Python renderers, and a preview server. self-hosted orchestration with external API integrations.
